Watch Party Overlay Timer - OBS 방송을 위한 비디오 오버레이 확장 프로그램

📺 주요 기능
Watch Party를 진행하거나 비디오 컨텐츠를 방송할 때, OBS Studio 등의 방송 프로그램에서 쉽게 캡처할 수 있는 오버레이를 제공합니다.

✨ 핵심 기능
• 실시간 비디오 정보 표시 (제목, 시리즈, 재생 시간)
• OBS에서 바로 사용 가능한 오버레이
• 스트리밍 디스플레이 창으로 원격 컴퓨터에서도 사용 가능
• VDO.Ninja 연동으로 네트워크를 통한 오버레이 전송 지원
• 비디오 플레이어 하단 바 스타일의 모던한 UI (1200×100px)

🎯 지원 플랫폼
• Laftel (라프텔) - 애니메이션 스트리밍
• Chzzk (치지직) - 네이버 라이브 스트리밍
• YouTube - 전 세계 동영상 플랫폼
• Netflix - 글로벌 OTT 서비스

🎨 오버레이 모드
1. 일반 모드: 모든 정보를 표시 (시리즈, 에피소드, 시간, 플랫폼)
2. 컴팩트 모드: 에피소드와 시간만 간결하게 표시
3. 그린스크린 모드: 크로마키 처리를 위한 녹색 배경

🚀 스트리밍 디스플레이 (v3.0+)
• 별도 창에서 오버레이 정보 표시
• 방송용 컴퓨터에서 원격으로 오버레이 사용 가능
• VDO.Ninja를 통한 실시간 스트리밍
• 플레이어 하단 바 스타일의 깔끔한 UI
• 동적 창 크기 조절 (기본: 600×100px, 설정 확장 시: 600×500px)
• 튜토리얼 시스템으로 첫 사용자 가이드 제공

🎨 고급 커스터마이징 (v3.1.0)
• 진행바 크기 조절: 너비(80~200px), 높이(2~12px) 개별 설정
• 창 크기 조절: 너비 300~1200px 범위에서 실시간 조절
• iOS 스타일 토글: 진행바/플랫폼 배지 표시 제어
• 플랫폼별 색상 배지: 각 사이트마다 고유한 브랜드 색상 적용
• 동적 레이아웃: 폰트 크기에 따라 UI 요소들이 자동 조정
• 색상 커스터마이징: 배경, 텍스트, 강조색, 진행바 색상 개별 설정

⚙️ 세부 기능
• 오버레이 위치 조절 (좌측/우측)
• 시간 표시 형식 선택 (현재/전체, 현재만, 숨김)
• 제목 표시 옵션 (시리즈+에피소드, 에피소드만, 숨김)
• 테마 선택 (라이트, 다크, 그린스크린)
• 폰트 크기 개별 조절 (에피소드, 시리즈, 시간: 6~40px)
• 오버레이 너비/여백 세밀 조절
• 설정 저장/불러오기/초기화

💡 사용 방법
1. 지원되는 비디오 사이트에서 동영상 재생
2. 확장 프로그램 아이콘 클릭
3. "정보 가져오기" 토글 ON
4. "오버레이 표시" 토글 ON으로 화면에 오버레이 표시
5. OBS에서 브라우저 화면 캡처

🔧 스트리밍 디스플레이 사용법 (투컴 환경)
1. "📺 스트리밍 디스플레이 열기" 버튼 클릭
2. 새 창에서 하단 화살표 클릭하여 설정 패널 확장
3. "🎥 VDO.Ninja 열기" 버튼으로 VDO.Ninja 사이트 이동
4. "Remote Screenshare into OBS" 선택 → 이 탭 선택 → Audio 해제 → Share
5. 생성된 링크를 방송 컴퓨터 OBS Browser Source에 추가

📚 튜토리얼 시스템
• 첫 사용자를 위한 4단계 가이드 제공
• 스트리밍 디스플레이 열기 시 자동으로 튜토리얼 시작
• "사용법 가이드 다시 보기" 버튼으로 언제든 재확인 가능
• 각 기능별 상세한 설명과 사용 팁 제공

📝 참고사항
• 페이지 새로고침 후에도 설정이 유지됩니다
• 동일 사이트 내에서 페이지 이동 시 자동으로 정보가 업데이트됩니다
• 각 사이트의 UI 변경에 따라 일시적으로 작동하지 않을 수 있습니다
• 모든 설정 변경이 실시간으로 반영되며 자동 저장됩니다

⚠️ 중요 안내사항
• 이 확장 프로그램은 Chrome 브라우저에 최적화되어 있습니다.
• Edge, Firefox 등 다른 브라우저에서는 정상 작동을 보장하지 않습니다.
• Chrome 버전 100 이상을 권장합니다.
