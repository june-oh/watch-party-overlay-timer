## [3.1.0] - 2025-07-09

### 추가된 기능 (Added)
- **진행바 크기 조절 기능**: 진행바의 너비(80~200px)와 높이(2~12px)를 개별적으로 조절할 수 있습니다.
  - 진행바 너비 조절로 에피소드 제목 영역과의 공간 배분을 사용자가 직접 설정 가능
  - 진행바 높이 조절로 시각적 강조 정도를 사용자가 조정 가능
  - 진행바 크기 변경 시 실시간으로 반영되며 자동 저장됩니다
- **스트리밍 디스플레이 너비 조절**: 창의 너비를 300px~1200px 범위에서 조절 가능하며, 실시간으로 창 크기가 변경됩니다.
- **첫 사용자를 위한 튜토리얼 시스템**: 스트리밍 디스플레이를 처음 열 때 각 컴포넌트에 대한 4단계 가이드가 표시됩니다.
  - 오버레이 정보 표시 설명
  - 설정 패널 열기 방법 (하단 화살표 바 클릭)
  - 모양 커스터마이징 기능 소개
  - OBS 연동 방법 안내
  - "다시 보지 않기" 옵션으로 튜토리얼 스킵 가능
  - 설정에서 "사용법 가이드 다시 보기" 버튼으로 언제든 재확인 가능
- **iOS 스타일 토글 버튼**: 진행바와 플랫폼 배지 표시를 개별적으로 제어할 수 있는 세련된 토글 스위치를 추가했습니다.
  - 진행바 표시/숨김 토글
  - 플랫폼 배지 표시/숨김 토글
  - 컴팩트 모드에서는 자동으로 두 요소 모두 숨김 처리
- **진행바 색상 독립 설정**: 진행바의 색상을 강조색과 별도로 설정할 수 있어 더 세밀한 커스터마이징이 가능합니다.
- 플랫폼별 고유 색상을 가진 배지 시스템 구현 (유튜브, 치지직, 라프텔, 넷플릭스 등)
- 확장 가능한 플랫폼 감지 시스템으로 새로운 플랫폼 쉽게 추가 가능
- 플랫폼 배지 호버 효과 및 색상 반전 애니메이션

### 개선된 점 (Improved)
- **스트리밍 디스플레이 색상 대비 자동 조정**: 배경색에 따라 텍스트, 드롭다운, 입력 필드의 색상이 자동으로 조정되어 가독성이 향상됩니다.
  - 밝은 배경에서는 어두운 텍스트, 어두운 배경에서는 밝은 텍스트로 자동 전환
  - 드롭다운 메뉴와 입력 필드의 배경색과 테두리색도 자동 조정
- **모서리 설정 시 투명 배경 자동 적용**: 모서리(border-radius)를 0보다 크게 설정하면 배경 투명도가 자동으로 0%로 설정되어 크로마키 사용에 최적화됩니다.
- **너비 조절 실시간 반영 개선**: 설정 패널이 열려있을 때도 너비 슬라이더 조정이 즉시 창 크기에 반영되어 실시간으로 크기 변화를 확인할 수 있습니다.
- **튜토리얼 가독성 향상**: 튜토리얼 표시 시 창 크기가 자동으로 800×700px로 확대되어 내용을 명확하게 볼 수 있으며, 튜토리얼 종료 후 원래 크기와 패널 상태로 자동 복원됩니다.
  - 튜토리얼 활성화 중에는 다른 기능들의 창 크기 변경이 차단되어 안정적인 크기 유지

### 변경된 점 (Changed)
- **팝업 UI 구조 개선**: 비디오 정보 아래에 "오버레이 표시 옵션"이라는 접을 수 있는 설정창으로 모든 오버레이 관련 설정을 통합
- **설정 구성 재편**: 모드, 위치, 제목 표시, 시간 표시, 테마, 사이트 표시 등 모든 오버레이 설정을 하나의 섹션으로 통합
- **고급 설정 통합**: 기존 고급 설정을 오버레이 표시 옵션 내 "세부 설정" 섹션으로 통합하여 추가 접기 없이 한 번에 모든 설정 접근 가능
- **플랫폼 배지 스타일 개선**: 호버 효과 제거 및 반투명 배경으로 변경하여 배경색과 조화롭게 개선

### 추가된 기능 (Added)
- **톱니바퀴 아이콘과 접기/펼치기 기능**: 명시적인 톱니바퀴 아이콘으로 설정창임을 표시하고 클릭으로 접기/펼치기 가능
- **세부 설정 구분**: 기본 설정과 세부 설정을 구분선과 소제목으로 명확히 분리
- **애니메이션 효과**: 설정창 펼치기/접기 시 부드러운 애니메이션 효과 적용

### 수정된 버그 (Fixed)
- **기본값 재설정 시 너비가 즉시 적용되지 않는 문제 해결**: 재설정 시 창 크기를 즉시 변경하고 약간의 지연 후 다시 업데이트하여 확실한 적용 보장
- **스트리밍 디스플레이 배경 투명도 문제 해결**: 모서리 설정 시 뒷 배경이 검은색으로 보이던 문제를 해결하여 완전 투명 배경이 적용됩니다.
- **iOS 토글 스위치 너비 문제 수정**: .setting-row label의 min-width: 60px 상속 문제 해결
- **스트리밍 디스플레이 열기 시 팝업에서 정보 가져오기 상태가 OFF로 표시되는 문제 수정**: 정보 가져오기 시작 시 팝업에 즉시 상태 업데이트 전송
- **너비/높이 슬라이더 조작 개선**: 슬라이더 드래그 중 창 크기가 계속 변경되던 문제를 해결하여 300ms debounce 처리로 드래그 완료 후에만 창 크기가 변경됩니다.
- **플레이어 정보창 배경색 문제 해결**: 사용자가 흰색으로 설정해도 회색으로 표시되던 문제를 해결하여 설정한 배경색이 정확히 반영됩니다.
- **설정창 닫기 시 너비 변경 문제 해결**: 설정창을 닫을 때(윗 화살표 클릭) 너비가 임의로 변경되던 문제를 해결하여 사용자가 설정한 너비가 유지됩니다.

### 수정된 버그 (Fixed)

- 슬라이더 조작 시 실시간 설정 저장 기능 추가

- 기본값으로 재설정 버튼이 작동하지 않던 문제 해결

- 높이 슬라이더가 실제 창 크기에 반영되지 않던 문제 해결

### 개선된 점 (Improved)

- 입력 필드와 버튼들의 통일된 어두운 스타일링
- 튜토리얼 시작 시 자동으로 정보 가져오기 시작
- 튜토리얼 설명 개선 (투컴 환경 VDO.Ninja 사용법 추가)
- 기본값 재설정 시 성공 메시지 표시

- 슬라이더와 수치 표시를 가깝게 배치하여 실시간 값 확인 용이
- iOS 토글 버튼의 정확한 크기와 애니메이션으로 사용성 향상
- 진행바 숨김 시 에피소드/시리즈 제목이 전체 영역을 사용하여 완전히 표시
- 너비 확장 시 제목 영역이 늘어나고 진행바와 시간 표시는 고정 크기 유지
- 배경 설정을 "배경색"과 "불투명도"로 분리하여 더 직관적인 UI 제공
- 반응형 디자인으로 작은 너비에서도 요소들이 적절히 표시되도록 개선

## [3.0.1] - 2024-12-22

### 개선된 점 (Improved)
- **스트리밍 디스플레이 기능 개선**
  - 스트리밍 창 열기 시 정보 가져오기만 자동 시작 (오버레이 표시는 독립적으로 유지)
  - 정보 가져오기 시작 후 1초 대기하여 정보를 받은 뒤 창이 열리도록 개선
  - 디스플레이 모드 변경 시 창 크기 자동 조정 (일반: 600x140px, 컴팩트: 450x120px)
  - 플랫폼 배지에 둥근 모서리(border-radius: 8px) 적용으로 시각적 개선
  - 컴팩트 모드에서 세로 중앙 정렬 개선
  - 창 높이를 10px 증가시켜 하단 화살표가 잘리지 않도록 수정
  - 1초 딜레이 동안 로딩 스피너와 "스트리밍 디스플레이 준비 중..." 메시지 표시
  - 로딩 중 배경 블러 효과로 시각적 피드백 향상
- **스트리밍 디스플레이 UX 개선**
  - 톱니바퀴 버튼 대신 하단 전체 너비의 확장 바 추가
  - 중앙에 위치한 화살표로 설정 패널 열기/닫기가 직관적으로 변경됨
- **YouTube 지원 개선**: YouTube 비디오를 시리즈 없는 단일 콘텐츠로 적절히 처리하도록 로직 개선
- **스트리밍 디스플레이 배경 처리 개선**
  - body 배경을 검은색으로 설정하여 투명 영역으로 인한 흰색 배경 노출 문제 해결
  - 오버레이 컨테이너 높이를 콘텐츠 크기에 맞게 최적화
  - VDO.Ninja 사용법 한국어 상세 가이드 추가
- **정보 가져오기 자동화**
  - 오버레이 표시 시 정보 가져오기 자동 시작
  - 스트리밍 디스플레이 창 열기 시 정보 가져오기 자동 시작
  - 사용자가 수동으로 정보 가져오기를 켤 필요가 없도록 개선
- **스트리밍 디스플레이 UI 개선**
  - 설정이 먼저 표시되고 도움말은 접을 수 있는 형태로 변경
  - 불필요한 Method 1 (Chrome Extension URL 복사) 제거
  - 연결 상태 표시 점(status dot) 제거
  - 플랫폼/사이트 이름 배지 유지
  - 창 너비를 450px에서 600px로 확대하여 가독성 향상
  - 모든 UI 텍스트를 한글로 번역
  - 드롭다운 메뉴 배경색 문제 해결 (검은색 배경 적용)
  - 기본 모드와 컴팩트 모드 추가:
    - 기본 모드: 2행 레이아웃 (1행: 시리즈 제목, 2행: 에피소드, 진행바, 시간, 플랫폼, 설정)
    - 컴팩트 모드: 1행 레이아웃 (에피소드, 시간(설정가능), 설정버튼만 표시)

### 변경된 점 (Changed)
- **VDO.Ninja 도움말 수정**: 투명 배경 지원 관련 내용 제거 (VDO.Ninja가 투명을 지원하지 않음)
- **스트리밍 디스플레이 창 공유 최적화**
  - 팝업 창으로 다시 변경하여 정확한 크기 제어 가능
  - 창 공유 시 여백 제거를 위해 모든 padding, border-radius, box-shadow 제거
  - 기본 창 크기를 450x80px로 조정하여 오버레이 크기에 최적화
  - 설정 패널 확장 시 450x600px로 자동 크기 조정
- **스트리밍 디스플레이 창 크기 조정**
  - Chrome 브라우저의 최소 창 높이 제한을 고려하여 기본 높이를 80px에서 100px로 변경
  - 좌우 padding 10px 추가로 버튼이 잘리지 않도록 개선
  - 창 높이 슬라이더의 최소값을 100px로 조정

### 수정된 버그 (Fixed)
- **스트리밍 디스플레이 시리즈 제목 표시 문제 해결**: 900px 미만 창 크기에서 시리즈 제목이 숨겨지는 미디어 쿼리 제거
- **스트리밍 디스플레이 화살표 표시 문제 해결**: 
  - 하단 바 높이를 12px로 증가하여 화살표가 잘리지 않도록 수정
  - CSS 회전 대신 JavaScript 텍스트 변경으로 화살표 방향 전환 안정화
- **Laftel 사이트 구조 변경 대응**: 사이트 UI 업데이트로 인한 정보 수집 실패 문제 해결
  - 클래스명 해시값 변경에 대응하는 유연한 selector 시스템 구현
  - 배열 형태의 다중 selector 지원으로 fallback 메커니즘 강화
  - 정보 컨테이너 검색 로직 개선으로 안정성 향상
  - 에피소드 정보를 "X화" 패턴으로 자동 감지하는 로직 추가
  - selector가 실패해도 패턴 매칭으로 정보를 찾는 이중 안전장치 구현
- **스트리밍 디스플레이 UI 문제 해결**
  - 설정 패널이 표시되지 않던 문제 수정 (collapsed 클래스 로직 반전)
  - 창 내용이 가로로 잘리던 문제 수정 (100vw → 100% 변경)
